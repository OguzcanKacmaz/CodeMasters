# Medium.com Clone Projesi: Redis, JWT ve React ile Backend ve Frontend Entegrasyonu

ASP.NET 6 Web API,JWT ve Redis kullanarak oluşturduğum backend ile React teknolojisiyle entegre edilmiş frontend içeren, Medium.com sitesinin klonunu geliştirdim. Amacım, bu projede yeni öğrendiğim teknolojileri uygulayarak backend ve frontend arasında ki etkileşimi sağlamak.

## Proje Özeti

Medium.com, içerik oluşturucuların ve okuyucuların bir araya geldiği popüler bir platformdur. İçerik oluşturucular, yazılar, hikayeler ve fikirleri paylaşırken, okuyucular ise ilgi çekici içerikleri keşfetmek için siteyi kullanır. Ben de bu başarılı platformun temel işlevselliğini yakalayıp ve mevcut bilgilerim üzerine yeni öğrendiğim teknolojileri de harmanlayarak kendimi geliştirmek amacı ile projeyi geliştirdim.

## Teknolojiler ve İşlevler

### Backend Geliştirme - ASP .NET 6 Web API ve Redis

Backend kısmını ASP .NET 6 Web API kullanarak oluşturdum. Kullanıcı kaydı ve girişi, yazı oluşturma gibi temel işlevleri sağlamak için API'lar geliştirdim.
Ayrıca, projenin performansını artırmak ve kullanıcı deneyimini geliştirmek için Redis kullandım. Redis, önemli verileri bellekte saklayarak veritabanı erişim sürelerini azaltır ve böylece uygulamanın hızlı çalışmasını sağlar.

### Güvenlik - JSON Web Token (JWT)

Kullanıcı kimlik doğrulama ve yetkilendirme için JSON Web Token (JWT) kullanarak projenin güvenliğini sağladım. Kullanıcıların API'larımıza güvenli bir şekilde erişmelerini sağlayarak, yetkisiz erişimi engelledim. Ayrıca, kullanıcı oturumlarını yönetmek ve güvenliği artırmak için JWT'leri kullanım süreleri belirledim.

### Frontend Geliştirme - React

Projemin kullanıcı arayüzü için modern ve etkileyici bir deneyim sağlamak için React'i tercih ettim. React'in bileşen tabanlı yapısı sayesinde, kullanıcı arayüzünü modüler bir şekilde tasarladım. Ayrıca, React'in hızlı yeniden-render özelliği sayesinde, kullanıcıların akıcı bir gezinti deneyimi yaşamalarını sağladım.

### Kullanılan Teknoloji ve Kütüphaneler
- JWT (JSON Web Token) tabanlı kimlik doğrulama ve yetkilendirme işlemleri
- Code First yaklaşımı
- Asenkron programlama
- Dependency Injection kullanarak bileşen tabanlı geliştirme
- Caching işlemleri
- Redis kullanımı
- Cross-Origin Resource Sharing (CORS) politikalarının yönetimi
- Entity Framework ile veritabanı işlemleri
- Repository design pattern kullanımı
- UnitOfWork design pattern kullanımı
- Option design pattern kullanımı
- N-Tier architecture
- Mapping işlemleri için AutoMapper kütüphanesi kullanımı
- RESTful API tasarımı ve geliştirilmesi
- Rol tabanlı yetkilendirme işlemleri
- Verileri sunucu tarafındaki Api den çekmek için Axios kütüphanesi kullanımı
- React-Router-Dom kütüphanesi kullanımı
- Context Api kullanımı
- Giriş ve kayıt işlemlerinde form verilerini doğrulamak için Formik ve Yup kütüphanesi kullanımı
- Material UI kullanımı
- Bootstrap-Icon kullanımı
- Sunucu tarafından gelen Json Web Token'ın decode işlemi için jwt-decode kütüphanesi kullanımı
- Verileri cookie de saklamak için react-cookie kütüphanesi kullanımı
- useState - useContext-useEffect gibi hookların kullanımı

### Öğrendiklerim ve Sonuç
Bu projeyi geliştirirken birçok yeni teknoloji ve kavramla tanıştım. Özellikle ASP .NET 6'nın gücü, React'in esnekliği ve Redis'in performans artırıcı etkisi beni etkiledi. JWT kullanarak güvenlik konusundaki bilgilerimi artırdım ve SPA mimarisini anlamak, frontend geliştirmeyi daha keyifli hale getirdi.

Sonuç olarak, bu projeyi geliştirerek hem backend hem de frontend alanında bilgilerimi pekiştirdim. Bir konsepti anlamak ve uygulamak, teoriden pratiğe geçişi sağlamanın en etkili yoludur. Medium.com klonu projem, hem yeni başlayanlar hem de deneyimli geliştiriciler için faydalı bir öğrenme deneyimi sunabilir.

Projemi geliştirirken karşılaştığım zorluklar beni daha iyi bir geliştirici olmaya teşvik etti ve bu süreçte sabır, azim ve öğrenmeye olan inancımı pekiştirdi. Kendimi geliştirmeye devam edeceğim ve gelecekte daha karmaşık ve inovatif projeler geliştirmek için bu deneyimleri kullanacağım.

Siz de yeni teknolojileri öğrenmek ve kendi projelerinizi geliştirmek için cesaretli adımlar atın! Yeni bilgiler edinmek, sınırlarınızı genişletmek ve hayal gücünüzü gerçeğe dönüştürmek için teknoloji dünyasının sonsuz potansiyelinden yararlanın.

Sağlam adımlar ve kodlamalar dilerim!
